📖 Janhavi's Blog – HTML & CSS Project
🚀 Overview

This project is a responsive and visually appealing blog homepage designed using HTML and CSS. It showcases different blog categories like travel, productivity, sustainability, and wellness in a card-based layout.

The goal of this project is to create a modern UI that is clean, user-friendly, and visually engaging.

✨ Features
🧩 Card-based layout for blog posts
🎨 Clean and minimal UI design
📱 Responsive design (works on different screen sizes)
🖼️ Image-based blog previews
👤 Author and date display for each post
🏷️ Category tags for better organization
🛠️ Tech Stack
HTML5 – Structure of the webpage
CSS3 – Styling and layout design
